# Ali PlayStation Store - UTS Pemrograman Web 2

### Deskripsi Project
[cite_start]Aplikasi E-Commerce bertema PlayStation yang dibangun menggunakan Fullstack JavaScript dan Tailwind CSS untuk memenuhi syarat UTS Pemrograman Web 2[cite: 2].

### Fitur Utama
* [cite_start]**Authentication (Simulasi)**: Login & Register dengan validasi email unik dan password minimal 6 karakter menggunakan LocalStorage[cite: 26, 27, 28, 31, 32].
* [cite_start]**Product Management**: Menampilkan list produk dari file JSON dengan struktur data yang lengkap (ID, Nama, Harga, Gambar, Deskripsi)[cite: 33, 34, 36].
* [cite_start]**Search System**: Fitur pencarian produk berdasarkan nama secara real-time[cite: 50, 51].
* [cite_start]**Cart System**: Tambah item, hapus item, dan perhitungan total harga otomatis[cite: 53, 54, 55, 57].
* [cite_start]**Checkout & Transaction**: Form checkout (Nama, Alamat, No HP) yang menghasilkan ID transaksi unik dan tersimpan di LocalStorage[cite: 58, 59, 65, 66].
* [cite_start]**Order History**: Menampilkan riwayat transaksi user beserta detailnya[cite: 67, 68, 69].
* [cite_start]**Responsive UI**: Desain menggunakan Tailwind CSS yang optimal di perangkat mobile maupun desktop[cite: 70, 71].

### Teknologi Wajib
* [cite_start]HTML5 [cite: 20]
* [cite_start]JavaScript ES6+ [cite: 21]
* [cite_start]Tailwind CSS [cite: 22]
* [cite_start]LocalStorage & JSON [cite: 23, 24]

### Cara Menjalankan
1. Clone atau download repository ini.
2. Buka `index.html` menggunakan browser atau Live Server di VS Code.
3. Pastikan koneksi internet tersedia untuk memuat Tailwind CSS.
